# 비전: Frank MVP11

> 생성일: 260423
> 진입점: MVP10 → MVP11

## 씨앗

**핵심 불편**: 실기기/시뮬레이터 테스트에서 발견된 결함 4개가 사용자 경험을 저해한다.
- 앱 첫 실행 시 항상 태그 로딩 실패 에러가 나타남 (BUG-001)
- 시뮬레이터/실기기 전환 시 서버 URL이 고정되어 연결 실패 (BUG-002)
- 즐겨찾기·오답 화면에 태그 필터가 없어 콘텐츠 누적 시 탐색 불편 (BUG-003)
- 피드에 개별 기사 대신 뉴스 인덱스 페이지(목록 페이지)가 노출됨 (BUG-004)

**욕구**: 새 기능 추가 전에 알려진 버그를 모두 수정하고, 뉴스 수집 품질을 실측 검증한다.

## 핵심 문제

실사용·테스트에서 발견된 4개 버그가 앱 기본 신뢰성과 콘텐츠 품질을 훼손하며, 다음 기능 개발 전에 반드시 해소해야 한다.

## 타겟 사용자

Frank 앱 개발자(자기 자신) — 실기기/시뮬레이터 전환 반복, 피드 품질 직접 확인.

## 성공 기준

- 앱 첫 실행 시 태그 로딩 에러 없이 정상 표시
- 시뮬레이터·실기기 전환 시 서버 URL 자동 분기로 연결 성공
- 즐겨찾기·오답 화면에서 태그 필터로 콘텐츠 탐색 가능
- 피드에서 뉴스 인덱스 페이지 노출 0건 (수집 품질 개선)
