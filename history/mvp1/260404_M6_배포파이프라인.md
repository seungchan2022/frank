## S1: Docker Compose 보완

- **목표**: 기존 docker-compose.yml에 헬스체크 + 재시작 정책 추가
- **수정 파일**: `docker-compose.yml`
- **상세**:
  - server: healthcheck (`/health`), `restart: unless-stopped`
  - web: healthcheck, `restart: unless-stopped`, depends_on server healthy
- **테스트**: `docker compose up -d` 후 `docker compose ps`로 healthy 확인

## S2: 배포 스크립트 (deploy.sh)

- **목표**: 요약 완료 시 ���로 요약된 ���사를 iMessage로 자동 전송
- **수신 대상**: 환경변수 `IMESSAGE_RECIPIENT` (이메일 주소)
- **트리거**: 서버 summarize API 내부, 요약 완료 직후
- **아키텍처**: 포트/어댑터 패턴 준수
  - `NotificationPort` trait 정의 (`domain/ports.rs`)
  - `ImessageAdapter` 구현 (`infra/imessage.rs`) — `std::process::Command`로 osascript 호출
