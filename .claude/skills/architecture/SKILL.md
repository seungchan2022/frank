# /architecture 스킬

코드베이스를 분석하여 D2 기반 고품질 아키텍처 다이어그램(SVG)을 자동 생성한다.

## 프로세스

1. **대상 선택**: 인자로 분석 대상 결정
2. **Serena 코드 분석**: 심볼/참조/의존성 추적
3. **Sequential Thinking 레이아웃 설계**: 요소 배치, 그룹화
4. **D2 다이어그램 생성**: SVG 렌더링

## MCP 도구

| MCP | 용도 |
|-----|------|
| Serena | 코드 심볼/참조 분석 |
| Sequential Thinking | 레이아웃 구조화 |
| D2 | 고품질 SVG 다이어그램 렌더링 |
