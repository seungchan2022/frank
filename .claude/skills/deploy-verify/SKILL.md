# /deploy-verify 스킬

서비스 배포 후 3단계 검증(헬스체크 → 스모크테스트 → 로그 확인)을 자동 실행한다.

## 프로세스

1. 서비스 식별
2. 배포 실행 (--skip-deploy 옵션 가능)
3. 헬스체크 (GET /health → 200 OK)
4. 스모크테스트 (핵심 엔드포인트 1~2개)
5. 로그 확인 (ERROR/CRITICAL 존재 여부)
6. 보고서 생성

## 제약

- 프로덕션 배포는 사용자 확인 후 실행
- FAIL 시 자동 롤백 금지
