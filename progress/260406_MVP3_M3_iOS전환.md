# M3: iOS 전환

> 상태: 📋 기획 완료
> 범위: 서버 + iOS
> 의존: M1

## 목표

iOS의 SupabaseArticleAdapter, SupabaseTagAdapter를 Rust API 기반 어댑터로 교체 + MVP2 부채 흡수.

## 전환 대상

| 현재 어댑터 | 메서드 | 전환 후 | Rust API |
|-----------|--------|--------|----------|
| SupabaseTagAdapter | fetchAllTags() | APITagAdapter | GET /api/tags |
| SupabaseTagAdapter | fetchMyTagIds() | APITagAdapter | GET /api/me/tags |
| SupabaseTagAdapter | saveMyTags() | APITagAdapter | POST /api/me/tags |
| SupabaseArticleAdapter | fetchArticles() | APIArticleAdapter | GET /api/me/articles |
| SupabaseArticleAdapter | fetchArticle() | APIArticleAdapter | GET /api/me/articles/:id |
| SupabaseAuthAdapter | updateOnboardingCompleted() | 수정 (Rust API 호출) | PUT /api/me/profile |

**유지 (변경 없음)**:
- SupabaseAuthAdapter (인증 전체 — signIn, signUp, signInWithApple, signOut, currentSession, getAccessToken)
- APICollectAdapter (이미 Rust API 사용)
