### S6. 웹: FeedItem 타입 + 카드 UI 업데이트

**변경 파일**: `web/src/lib/types/article.ts`, `web/src/routes/feed/+page.svelte`, `web/src/routes/favorites/+page.svelte`

`article.ts`:
```typescript
export interface FeedItem {
  // 기존 필드 유지
  image_url?: string | null;  // 추가
}
```

`feed/+page.svelte` 카드 레이아웃 변경:
```
Before: VStack (제목 + source + snippet)
After:  HStack {
          [썸네일 72×72 or 플레이스홀더]
          VStack { 제목 / source·발행일 }
        }
```
- `image_url` 있으면: `<img src={item.image_url} class="w-18 h-18 rounded object-cover flex-shrink-0" />`
- `image_url` 없으면: `<div class="w-18 h-18 rounded bg-gray-200 flex-shrink-0">` (플레이스홀더)
- `onerror` 핸들러: 로드 실패 시 플레이스홀더로 교체

`favorites/+page.svelte`: 동일 카드 컴포넌트 적용

### S9. iOS: ArticleCardView 레이아웃 변경

**변경 파일**: `ios/Frank/Frank/Sources/Components/ArticleCard/ArticleCardView.swift`

```
Before: VStack { Text(title) / HStack(source·발행일) }
After:  HStack(alignment: .top, spacing: 12) {
          // 썸네일 영역 72×72
          if let imageUrl = article.imageUrl {
            AsyncImage(url: imageUrl) { phase in ... }
              .frame(width: 72, height: 72)
              .clipShape(RoundedRectangle(cornerRadius: 8))
          } else {
            RoundedRectangle(cornerRadius: 8)
              .fill(Color(.systemGray5))
              .frame(width: 72, height: 72)
          }
          // 텍스트 영역
          VStack(alignment: .leading, spacing: 6) {
            Text(article.title).font(.headline)
            HStack { Text(source) · Text(relativeTime) }.font(.caption)
          }
        }
```

- `AsyncImage` phase: `.success` → 이미지 표시, 나머지 → 플레이스홀더 유지 (loading/failure 동일)
- FavoritesView도 동일 `ArticleCardView` 사용 → 자동 반영
