웹·앱 기능 싱크를 맞춘다.

## 구현 방향

- API는 이미 태그 데이터 반환 중 → 서버 변경 불필요
- iOS M2와 동일한 클라이언트 사이드 필터링 방식
- 피드에 태그 칩 필터 컴포넌트가 이미 있으면 재사용
- Svelte 5 `$state`로 `selectedTag` 관리, `$derived`로 필터된 목록 계산

```svelte
let selectedTag = $state<string | null>(null);
let filtered = $derived(
  selectedTag ? items.filter(i => i.tags?.includes(selectedTag)) : items
);
```
