## 서브태스크 상세

### ST-1: 태그 stale 이슈

**재현 절차:**
```
1. 태그 A, B 선택 → 기사 수집
2. 태그 A 삭제
3. 피드 확인 → 태그 A 기반 기사가 여전히 표시되는지 확인
```

**예상 해결 방향** (재현 후 분석으로 확정):
- 피드 조회 시 현재 사용자 태그와 기사 태그 교집합으로 필터링 (서버 쿼리 수정)
- 또는 태그 삭제 시 관련 기사 소프트 삭제 (DB cascade)

### ST-2: Supabase Manual Linking 조사

**조사 항목:**
